*) Command:
  -> ctrl + t (list of all available files)
  -> ctrl + r (list of history of commands)
  -> fzf (fuzzy finder)
  -> alt + c (change directory)
  -> cat <file_name> | fzf (search line in file)
  -> fzf-tmux (search in tmux)
  -> git branch | fzf (search in git)

  -> learn more: https://youtu.be/qgG5Jhi_Els
